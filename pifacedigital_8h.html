void pifacedigital_close ( uint8_t  hw_addr)

Closes a PiFace Digital with the specified hardware address (turns off interrupts, closes file descriptor if it is no longer needed).

Example: 

pifacedigital_close(0);
Parameters
hw_addrThe hardware address (configure with jumpers: JP1 and JP2).
uint8_t pifacedigital_digital_read ( uint8_t  pin_num)

Read from the input port on PiFace Digital 0. If you want to read from other ports or PiFace Digital's with other hardware addresses then use the more powerful function <pifacedigital_read_bit>"()".

Example (read from bit 6 of INPUT on PiFace Digital 0): 

pifacedigital_digital_read(6);

int pifacedigital_open ( uint8_t  hw_addr)

Opens and initialises a PiFace Digital with the specified hardware address. Returns a file descriptor for making raw SPI transactions to the MCP23S17 (for advanced users only).

Example: 

pifacedigital_open(0);
int pifacedigital_fd = pifacedigital_open(0); // advanced
Parameters
hw_addrThe hardware address (configure with jumpers: JP1 and JP2).
int pifacedigital_open_noinit ( uint8_t  hw_addr)

Opens a PiFace Digital with the specified hardware address without initialising it. Returns a file descriptor for making raw SPI transactions to the MCP23S17 (for advanced users only).

Example: 

pifacedigital_open_noinit(0);
int pifacedigital_fd = pifacedigital_open_noinit(0); // advanced
Parameters
hw_addrThe hardware address (configure with jumpers: JP1 and JP2).
